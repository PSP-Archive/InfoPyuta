はじめにお読みください

　InfoPyuta v0.02Jは，Windows上で動作する，空想上のぴゅう太エミュレータです．現時点では，商用ゲームなどは，動作しないと思います．最終的には，完全なエミュレーションを目指していますが，現状では，情報があまりにも欠落している状態です．そのためには，作者は，ソースコードを配布することで，皆様のご協力を仰ぐことにしました．実際，プロジェクトを開始する以前から，一人の力では，どうしようもないと思っていましたので，ソースコードの配布は，ある意味，予定通りのことです．
